Difference between localstorage and sessionstorage

1.sessionstorage works session based if we close the tab data will be disappeared, but localstorage does not work session based it need to deleted .
2.sessionstorage accessible from same tab, but localstorage accessible from any window.
3.sessionstorage capacity has 5Mbs , but localstorage capacity has 5-10 Mbs.
4. sessionstorage can auto expiry, localstorage can not auto expiry.
5. In sessionstorage data persistence is temporary and in localstorage data persistence is permanent.

How does javascript works

1.When we run our javascript code , firstly this code received by parser and it is check for syntactic error line by line.
2. When there are no error in the program it creates "abstract syntax tree".
3. After creating abstract syntax tree javascript engine converts the javascript code into the machine code.
4. Then the machine code is sent to the system for execution.
5. Finally the machine code run by the engine and we get our output result.
